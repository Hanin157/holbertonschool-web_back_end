# Python Async Function

This project focuses on asynchronous programming in Python using asyncio.

## Objectives

- Understand async and await syntax
- Execute async programs with asyncio
- Run concurrent coroutines
- Create asyncio tasks
- Use the random module

## Files

- `0-basic_async_syntax.py`: Coroutine that waits for a random delay
- `1-concurrent_coroutines.py`: Runs multiple coroutines concurrently
- `2-measure_runtime.py`: Measures average runtime
- `3-tasks.py`: Creates an asyncio task
- `4-tasks.py`: Runs multiple asyncio tasks concurrently
