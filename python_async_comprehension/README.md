# Python Async Comprehension

This project focuses on asynchronous generators and async comprehensions in Python.

## Objectives

- Understand async generators
- Use async for loops
- Learn async comprehensions
- Type annotate generators
## Requirements

- Python 3.9
- Ubuntu 20.04 LTS
- Code follows pycodestyle
- All files are executable
- All functions are type-annotated
- All modules and functions include proper documentation

## Files

- `0-async_generator.py`: Defines an asynchronous generator that yields random numbers
