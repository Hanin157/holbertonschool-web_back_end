# Python Variable Annotations

This project focuses on learning and applying type annotations in Python 3.

## Objectives

- Understand type annotations in Python
- Use annotations to specify function signatures and variable types
- Learn about duck typing
- Validate code using mypy

## Requirements

- Python 3.9
- Ubuntu 20.04 LTS
- Code follows pycodestyle
- All files are executable

## Files

- `0-add.py`: Function that adds two floats using type annotations
